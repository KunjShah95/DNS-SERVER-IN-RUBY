## Installation

To install the DNS Server, clone the repository and run the setup script:

```bash
git clone https://github.com/KunjShah95/dns-server.git
cd dns-server
./setup.sh
```

## Usage

To start the DNS Server, run the following command:

```bash
ruby dns_server.rb
```
